Gadfly.render — Method
Gadfly.Geom.render(geom::DensityCI, theme, aes)

We need to do all the rendering manually because Gadfly.Geom.density is a line element and therefore cannot do a ribbon at the same time. The way to get that normally would be via Gadfly.Stat.density with Geom.polygon. Unfortunately, the polygon doesn't work with subplot_grid.
